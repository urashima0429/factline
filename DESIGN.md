# Factorio 生産ラインレイアウト最適化ツール

Note: これは人間が指示を与えるためのドキュメントであり，AI Agentがこのドキュメントを直接編集することを禁止する．

## 目的

Factorio の生産ラインを制約充足問題 (CSP) + 最適化として定式化し，バスを左側に置いたレイアウトで **幅最小化** を行う．原則として「何が満たされるべきか（制約）」だけを記述し探索手順はソルバーに任せる．

---

## Model

- 盤面はタイル単位の格子 (grid) として表現される．
  - 空白，1x1 タイル，向きなし
  - 組立機， 3x3 タイル，流体入力がある場合のみ向きアリ
  - インサータ， 1x1 タイル，向きアリ
  - （地上）ベルト，1x1 タイル，向きとレーン（LEFT/RIGHT）アリ
  - 地下ベルト入口，1x1 タイル，向きとレーン（LEFT/RIGHT）アリ
  - 地下ベルト出口，1x1 タイル，向きとレーン（LEFT/RIGHT）アリ
  - 箱，1x1 タイル，向きなし
- モジュール範囲化技術（範囲内の機械に速度等のバフを与える）を考慮し，組立機の上/下2行のみを入出力に利用できるものとする
- 結果として，盤面高さは 2 + 3 + 2 = 7固定．

--- 

## Formulation
- Input 
  - バスから生産ラインへ入力されるアイテム集合
  - ~箱から生産ラインへ入力されるアイテム集合~ (今回は考慮しない)
  - 生産ラインからバスへ出力されるアイテム
  - 生産ラインから箱に格納されるアイテム
  - 地下ベルト最大距離 (`underground_max_distance`)
  - 生産ラインに配置される組立機の数と仕様（固形入力アイテム集合，流体入力0/1，出力アイテム1）
- Output
  - もし存在しない場合は **UNSAT** を出力．
  - もし存在する場合は最小幅レイアウト（2x2描画＋左ラベル）．
- Objective function: 盤面の幅を最小化（高さは固定）
- Variables:
  - 盤面幅
    - 下限: 組立機を隙間なく横に並べたときの幅
    - 上限: 下限 + 余白として3程度
  - 各タイルの種類（重複禁止，盤面幅を超える座標にはタイルを配置できない）
  - 各タイルの向き（向きを持たないタイル種別では無効）
  - 各タイルの各レーンにどのアイテムが流れるか（レーンを持たないタイル種別では無効）
  - バス入出力の位置
- Constraints:
  - 組立機制約
    - 存在制約: 指定されたスペックの組立機が盤面上に指定された数だけ存在する．
    - 入力制約: 各組立機の指定されたすべての入力は隣接するインサータを介して供給される．
    - 出力制約: 各組立機の指定された出力は隣接するインサータを介して排出される．
    - 流体入力制約: （任意，指定されたスペックに流体入力がある場合）各組立機の指定された流体入力は隣接するパイプから流体が供給される．
  - インサータ制約
    - 入力制約: 各インサータの入力は，隣接する供給タイル（ベルト，地下ベルト入口，地下ベルト出口，箱，組立機）から供給される．
    - 出力制約: 各インサータの出力は，隣接する排出タイル（ベルト，地下ベルト入口，地下ベルト出口，箱，組立機）に排出される．
      - 排出タイルがレーンを持つ場合（ベルト，地下ベルト入口，地下ベルト出口）遠い側のレーンに排出される．
  - バスI/O制約
    - 入力制約: 指定されたスペックのすべてのアイテムが盤面左端から供給される．
    - 出力制約: 指定されたスペックのアイテムが盤面左端に排出される．
  - 箱I/O制約
    - ~入力制約: 指定されたスペックのすべてのアイテムが盤面上の箱から供給される．~ (今回は考慮しない)
    - 出力制約: 指定されたスペックのすべてのアイテムが盤面上の箱に排出される．
  - ベルト制約
    - 入力制約: ベルトの各入力は，隣接する供給タイル（ベルト，地下ベルト出口）から供給される．
    - 出力制約: ベルトの出力は，自身の向きに隣接する排出タイル（ベルト，地下ベルト入口）が存在する場合には，そこに排出される．
  - 地下ベルト制約
    - 一対一対応する地下ベルト入口と地下ベルト出口は一直線上に配置されなければならない（地下で曲げてはいけない）．
    - 一対一対応する地下ベルト入口と地下ベルト出口の距離は `underground_max_distance` 以下でなければならない．
    - 一対一対応する地下ベルト入口と地下ベルト出口は向きが同じでなければならない．
    - 地下区間は重複してはいけない．すなわち，あるタイルは多くとも一つの地下区間にしか属せない．
      - ただし，縦方向の地下区間と横方向の地下区間は重複してもよい．
    - 地下ベルト入口制約  
      - 入力制約: 地下ベルト入口の各入力は，隣接する供給タイル（ベルト，地下ベルト出口）から供給される．
      - 出力制約: 地下ベルト入口の各出力は，一対一対応する排出タイル（地下ベルト出口）*が存在する場合には，そこ*に排出される．
    - 地下ベルト出口制約
      - 入力制約: 地下ベルト出口の各入力は，一対一対応する供給タイル（地下ベルト入口）*が存在する場合には，そこ*から供給される．
      - 出力制約: 地下ベルト出口の各出力は，隣接する排出タイル（ベルト，地下ベルト入口）に排出される．
  - レーン制約: 
    - 一つの連続するレーンには一つのアイテムのみが流れる．
